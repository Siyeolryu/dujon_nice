# 공무 용어 사전

종합건설 공사부 공무팀에서 쓰는 용어의 정의와 흔한 오용.
UI 문구·필드명·보고서 문장을 만들 때 이 사전에 맞춘다.

## 계약·금액

| 용어 | 정의 | 오용 주의 |
|---|---|---|
| 도급금액 | 발주처와 체결한 계약 금액 | "계약금"(선수금 뉘앙스)과 혼동 금지 |
| 실행예산(실행) | 회사 내부 목표 원가. "실행을 짠다" | 도급금액과 반대 개념으로 대비해 사용 |
| 변경계약(설계변경) | 공사 중 계약 내용·금액 변경 | "계약 수정" 대신 "변경계약" |
| 하도급/외주 | 협력업체에 재도급 | 공식 문서엔 "하도급", 구어로 "외주" |
| 관급 | 관공서 발주. 관급공사, 관급자재 | |
| 사급 | 시공사가 직접 조달하는 자재 | 관급자재의 반대 |
| 순공사비 | 재료비+노무비+경비 | |
| 일반관리비/이윤 | 원가 외 항목 | |

## 기성·수금

| 용어 | 정의 | 오용 주의 |
|---|---|---|
| 기성 | 공사 진척분에 대한 대금. "기성을 친다/올린다" | "인보이스", "청구서 발행" 금지 |
| 기성청구 | 진척분 대금을 발주처에 청구 | Event.type enum 값 |
| 기성률 | 도급 대비 기성 누계 비율 | |
| 수금 | 청구한 기성의 입금 확인 | "수납", "입금처리"보다 "수금" |
| 선급금 | 계약 후 선지급 받는 금액 | "선수금"(회계용어)과 구분 |
| 유보금/하자보증금 | 준공 후 일정 기간 예치 | |

## 적산·견적

| 용어 | 정의 |
|---|---|
| 적산 | 도면에서 수량(물량)을 산출하는 일 |
| 물량산출(서) | 적산의 산출 근거 문서 |
| 견적 | 수량 × 단가로 금액 산정 |
| 내역서 | 공종별 수량·단가·금액 표. "산출내역서" |
| 일위대가 | 단위 작업당 단가 구성표 |
| 품셈 | 표준 노무·자재 소요량 기준 |

## 공정·현장

| 용어 | 정의 |
|---|---|
| 착공/준공 | 공사 시작/완료 |
| 공정률 | 전체 대비 진행 비율 |
| 공사기간(공기) | 착공~준공 기간. "공기 연장" |
| 현장소장 | 현장 총괄 책임자. 줄여서 "소장" |
| 공무 | 현장·본사의 계약/기성/문서 행정 담당 |
| 감리 | 발주처 측 공사 감독 |
| 시공사/원청 | 우리 회사 위치 |

## 문서·행정

| 용어 | 정의 | 오용 주의 |
|---|---|---|
| 공문 | 기관·회사 간 공식 문서 | "오피셜 레터" 금지 |
| 시행문/접수문 | 발신하는/접수한 공문 | UI는 "발신"/"수신" 사용 |
| 문서번호 | `제YYYY-NNN호` 연번 | |
| 회신기한 | 답변해야 하는 기한 | "답장 기한" 대신 "회신기한" |
| 인허가 | 관공서 허가·신고 (착공신고, 사용승인 등) | |
| 민원 | 주민·이해관계자 요구 대응 | |
| 결재 | 상급자 승인 절차 | "결제"(지불)와 혼동 금지 — 흔한 오타 |
| 품의 | 결재를 올리는 행위. "품의서" | |
| 특이사항 | 보고서의 리스크·전달 사항 항목 | "이슈", "노트" 금지 |
