# 보고서 표준 양식 (주간보고 · 일일보고)

report.html의 자동 생성 로직과 인쇄 레이아웃이 따라야 하는 양식.
실무 피드백으로 양식이 바뀌면 이 문서를 먼저 고친다.

## 주간 업무보고

### 헤더

```
                        주 간 업 무 보 고
보고기간: 2026-07-13(월) ~ 2026-07-17(금)
소  속: 공사부 공무팀     직급: 대리     성명: 홍길동
```

- 우측 상단에 결재란 표 (담당 / 팀장 / 부장 — 3칸 기본, 조정 가능)
- 제목은 글자 간격을 띄운 "주 간 업 무 보 고" 형태가 관례

### 본문 — 현장별 그룹핑

```
■ ○○아파트 신축공사 현장
  1. 금주 실적
    - 7월 기성청구 서류 작성 완료 (07/15)
    - 하도급 변경계약(철근콘크리트) 진행 중 — 견적 대조 단계
  2. 차주 계획
    - 하도급 변경계약 품의 상신
    - 착공계 보완서류 제출 (기한 07/24)
  3. 특이사항
    - 발주처 공문(제2026-041호) 회신기한 07/22 — 회신대기
```

- 현장 구분 기호는 `■`, 항목은 `1. 2. 3.`, 세부는 `-` 개조식.
- 현장 미지정 업무는 마지막에 "■ 공통·본사 업무" 블록으로 묶는다.

### 문장 톤 규칙 (자동 생성 문구에 적용)

1. 개조식·명사형 종결: "~완료", "~진행 중", "~예정", "~상신"
2. 완료 업무: `{업무명} 완료 ({MM/DD})`
3. 진행 업무: `{업무명} 진행 중` (+ 필요 시 상태 보충)
4. 계획 업무: `{업무명}` 또는 `{업무명} (기한 {MM/DD})`
5. 특이사항 자동 수집 대상:
   - 회신대기 문서: `{발신처} 공문({문서번호}) 회신기한 {MM/DD} — 회신대기`
   - 기간 내 일정: `{일정명} ({MM/DD})`
   - 지연 업무: `{업무명} 지연 (기한 {MM/DD} 경과)`

## 일일 업무보고

주간보고의 축약형. 기간 대신 일자 하나.

```
                        일 일 업 무 보 고
보고일자: 2026-07-18(금)
소  속: 공사부 공무팀     직급: 대리     성명: 홍길동

1. 금일 실적
2. 명일 계획
3. 특이사항
```

- 현장 수가 적으면 현장별 그룹핑 없이 항목만 나열해도 된다.
- "금일/명일" 용어 사용 (오늘/내일 대신).

## 인쇄 규칙

- A4 세로, 여백 14mm/12mm, 본문 10~11pt, 1~2장 이내.
- 결재란은 인쇄에서만 표시 (`print.css`의 `.print-only`).
- 텍스트 복사 출력은 위 본문 구조를 평문으로 재현 (메신저·메일 붙여넣기용 — 표 없이).
