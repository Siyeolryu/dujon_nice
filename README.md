# dujon_nice — 공무팀 업무관리 플랫폼

종합건설 공사부 공무팀의 업무를 한곳에서 관리하는 웹 프로그램입니다.
설치 없이 브라우저에서 바로 사용합니다.

**바로가기: https://siyeolryu.github.io/dujon_nice/**

## 주요 기능

| 기능 | 설명 |
|---|---|
| 업무 보드 | 대기/진행/완료 칸반으로 내 업무 관리 |
| 주간·일일 업무보고 | 입력한 업무로 보고서 자동 생성 → 인쇄/복사 제출 |
| 공문·문서 관리대장 | 수신/발신 문서 대장, 회신기한 경고 |
| 일정 캘린더 | 기성청구·계약마감·회의 일정 + 업무 마감 합성 표시 |
| 현장·소장 배정현황 | 지도에서 현장별 소장 배정, 30km 거리 판정, 시/군 구분 |
| 도면 검토 | 도면 PDF 업로드, 축척 보정 후 면적·길이 측정, OCR, 차수 비교 |

## 데이터 안내

- 모든 데이터는 **사용자 브라우저에만** 저장됩니다 (서버 없음).
- 설정 화면에서 JSON 백업 내보내기/가져오기를 지원합니다. 주 1회 백업을 권장합니다.
- 지도 기능은 카카오맵 JavaScript 키를 설정 화면에 입력하면 활성화됩니다.
