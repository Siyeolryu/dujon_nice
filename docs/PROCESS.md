# 프로세스 플로우 (PROCESS)

> 누구나(상급자 포함) 보고 이해할 수 있는 업무 흐름도.
> 이 플랫폼이 공무팀 업무를 어떻게 처리하는지 5개 흐름으로 설명한다.

## 1. 일일 업무 흐름 — 등록에서 주간보고까지

매일 입력한 업무가 금요일 주간보고로 자동 정리된다. 보고서를 따로 쓰지 않는다.

```mermaid
flowchart LR
    A["아침<br/>대시보드 확인"] --> B["오늘 마감·<br/>진행 업무 파악"]
    B --> C["업무 등록/<br/>상태 변경"]
    C --> D["완료 처리<br/>(완료일 자동 기록)"]
    D --> E["금요일<br/>주간보고 자동 생성"]
    E --> F["문구 다듬기<br/>(인라인 편집)"]
    F --> G["인쇄(A4·결재란)<br/>또는 텍스트 복사 제출"]
```

## 2. 문서 처리 흐름 — 공문 접수에서 회신까지

```mermaid
flowchart LR
    A["공문 접수/발송"] --> B["대장 등록<br/>(문서번호 자동 제안)"]
    B --> C["담당·현장 지정"]
    C --> D["회신기한 입력<br/>→ 캘린더 자동 표시"]
    D --> E{"기한 임박?"}
    E -- "예" --> F["대장·대시보드에<br/>경고 강조"]
    E -- "아니오" --> G["처리/회신"]
    F --> G
    G --> H["완료 처리"]
```

## 3. 도면 검토 흐름 — 업로드에서 적산 기초자료까지

```mermaid
flowchart LR
    A["도면 PDF 업로드<br/>(현장·공종·차수 태그)"] --> B["축척 보정<br/>(두 점 + 실제 거리)"]
    B --> C["면적·길이 측정<br/>(화면에서 점 찍기)"]
    C --> D["측정 결과 저장<br/>(라벨: 예. 101호 거실)"]
    D --> E["측정 목록<br/>XLSX 내보내기"]
    E --> F["적산·견적<br/>기초자료 활용"]
    A -.-> G["차수 간 오버레이 비교<br/>(불일치 부위 시각 확인)"]
    A -.-> H["OCR 텍스트 추출<br/>(치수·실명·표)"]
```

## 4. 현장소장 배정 흐름 — 30km 판정

```mermaid
flowchart LR
    A["신규 현장 등록"] --> B["주소 검색<br/>(좌표·시/군 자동 저장)"]
    B --> C["지도에서 소장별<br/>현재 배정 현장 확인"]
    C --> D["현재 현장 기준<br/>신규 현장까지 거리 계산"]
    D --> E{"30km 미만?"}
    E -- "예" --> F["배정 가능 (초록)"]
    E -- "아니오" --> G["원거리 표시"]
    F --> H["배정 등록<br/>(배정 표 갱신)"]
```

## 5. 데이터 관리 흐름 — 백업과 플랫폼 전환

```mermaid
flowchart LR
    A["평상시<br/>브라우저에 자동 저장"] --> B["주 1회<br/>JSON 백업 내보내기"]
    B --> C["백업 파일 보관<br/>(개인 드라이브)"]
    C --> D["필요 시 가져오기로<br/>완전 복원"]
    C -.-> E["연말: 백업 파일 그대로<br/>Supabase DB 시딩"]
    E -.-> F["공사부 멀티유저<br/>플랫폼 전환"]
```

## 배포 전 검증 체크리스트 (RULES.md §6)

1. 전 페이지 콘솔 에러 0건 (빈 상태 포함)
2. 업무/문서/일정/현장 CRUD → 새로고침 후 유지
3. JSON 백업 내보내기 → 초기화 → 가져오기 → 완전 복원
4. 주간보고 A4 인쇄 미리보기 잘림 없음 + 텍스트 복사 정상
5. 모바일 폭(390px) 사이드바·테이블 동작
6. 상대 경로만 사용 (`/assets/...` 검색해서 0건)
7. push 후 Pages URL에서 재확인
8. 도면: PDF 업로드 → 축척 보정 → 면적 측정값 수기 계산과 일치, 새로고침 후 유지
9. 지도: 현장 2곳 등록 → 거리·시/군 표시, 30km 판정 색상, 키 미입력 시 안내 문구
